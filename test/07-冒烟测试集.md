## 1. 冒烟测试概述

### 1.1 冒烟测试定义

冒烟测试（Smoke Testing）是对软件基本功能进行快速验证的测试方法，目的是确认软件的主要功能能够正常运行，如果主要功能存在问题，则不需要进行更深入的测试。

### 1.2 冒烟测试目的

1. **快速验证**: 快速验证系统基本功能是否正常
2. **质量把关**: 确保版本发布前核心功能可用
3. **效率提升**: 减少无效测试，提高测试效率
4. **风险控制**: 及早发现严重问题，降低发布风险

### 1.3 冒烟测试原则

1. **覆盖核心功能**: 只测试最核心、最重要的功能
2. **执行快速**: 测试用例数量少，执行时间短
3. **结果明确**: 测试结果容易判断，通过/失败清晰
4. **自动化优先**: 优先实现自动化执行

## 4. 冒烟测试脚本

### 4.1 脚本代码

```python
#!/usr/bin/env python3
"""
龙虾办公室 - 冒烟测试脚本

快速验证系统核心功能
"""

import pytest
import requests
import time
import concurrent.futures

BASE_URL = "http://localhost:8000"
FRONTEND_URL = "http://localhost:5173"
TIMEOUT = 10


class TestSmokeAPI:
    """冒烟测试 - API测试"""
    
    @pytest.mark.smoke
    def test_health_check(self):
        """ST-001: 系统健康检查"""
        url = f"{BASE_URL}/health"
        response = requests.get(url, timeout=TIMEOUT)
        
        assert response.status_code == 200
        data = response.json()
        assert data.get("status") == "healthy"
    
    @pytest.mark.smoke
    def test_get_agents_status(self):
        """ST-002: 获取Agent状态列表"""
        url = f"{BASE_URL}/api/v1/agents/status"
        response = requests.get(url, timeout=TIMEOUT)
        
        assert response.status_code == 200
        data = response.json()
        
        assert "agents" in data
        assert "summary" in data
        
        summary = data["summary"]
        assert "total" in summary
        assert "idle" in summary
        assert "conversing" in summary
        assert "working" in summary
    
    @pytest.mark.smoke
    def test_get_agent_tasks(self):
        """ST-003: 获取Agent任务清单"""
        url = f"{BASE_URL}/api/v1/agents/tasks"
        response = requests.get(url, timeout=TIMEOUT)
        
        assert response.status_code == 200
        data = response.json()
        
        assert isinstance(data, dict)
    
    @pytest.mark.smoke
    def test_get_requests_stats(self):
        """ST-004: 获取请求统计数据"""
        url = f"{BASE_URL}/api/v1/requests/stats"
        response = requests.get(url, timeout=TIMEOUT)
        
        assert response.status_code == 200
        data = response.json()
        
        assert "hourly" in data or "total" in data
    
    @pytest.mark.smoke
    def test_get_token_stats(self):
        """ST-005: 获取Token统计数据"""
        url = f"{BASE_URL}/api/v1/tokens/stats"
        response = requests.get(url, timeout=TIMEOUT)
        
        assert response.status_code == 200
        data = response.json()
        
        assert "total_used" in data or "budget" in data


class TestSmokePerformance:
    """冒烟测试 - 性能测试"""
    
    @pytest.mark.smoke
    def test_api_response_time(self):
        """ST-006: API响应时间验证"""
        url = f"{BASE_URL}/api/v1/agents/status"
        
        start_time = time.time()
        response = requests.get(url, timeout=TIMEOUT)
        end_time = time.time()
        
        response_time = end_time - start_time
        
        assert response.status_code == 200
        assert response_time < 1.0, f"响应时间 {response_time:.2f}s 超过1秒限制"
    
    @pytest.mark.smoke
    def test_concurrent_requests(self):
        """ST-007: 并发请求处理"""
        url = f"{BASE_URL}/api/v1/agents/status"
        
        def make_request():
            response = requests.get(url, timeout=TIMEOUT)
            return response.status_code
        
        with concurrent.futures.ThreadPoolExecutor(max_workers=10) as executor:
            futures = [executor.submit(make_request) for _ in range(10)]
            results = [f.result() for f in concurrent.futures.as_completed(futures)]
        
        assert all(status == 200 for status in results), "存在失败的并发请求"


if __name__ == "__main__":
    pytest.main([__file__, "-v", "--tb=short", "-m", "smoke"])
```
