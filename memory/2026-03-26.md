## 🔧 技术要点

### 时间处理
```python
# UTC 时间戳 → 北京时间
created_at = datetime.utcfromtimestamp(ts_float) + timedelta(hours=8)

# ISO 格式 → 北京时间
utc_dt = datetime.fromisoformat(ts.replace('Z', '+00:00'))
beijing_dt = utc_dt + timedelta(hours=8)
```

### Agent 识别
```python
# 优先使用目录名作为 agent_id（最准确）

```
