## 📋 **任务命名规范**

### **基本原则**

所有任务名称必须以**归属 Agent** 开头，格式：
```
{agent_id} {任务描述} - {执行频率}
```

### **命名示例**

| Agent | 正确示例 | 错误示例 |
|-------|---------|---------|
| **work-claw** | `work-claw AI 新闻日报 - 每日早上 7 点` | `AI 新闻日报 - 每日早上 7 点` ❌ |
| **work-claw** | `work-claw 晨间天气预报 - 每日 6:30` | `晨间天气预报 - 每日 6:30` ❌ |
| **daughter** | `daughter 每日汇报 - 20:30` | `每日汇报 - 20:30` ❌ |
| **dev-claw** | `dev-claw 心跳 - 每日 8/12 点` | `心跳 - 每日 8/12 点` ❌ |
| **main** | `main 工作日志提醒-main` | `工作日志提醒-main` ❌ |

## 🤖 **Cron 任务创建规范**

### **自动添加 Agent 前缀**

**文件**: `backend/cron_sync.py`

**逻辑**：
```python
# 1. 尝试从任务名解析归属 agent
owner = parse_owner_from_task_name(task_name)

# 2. 如果没有前缀，根据任务类型自动添加
if not owner:
    if '日报' in task_name or '天气' in task_name or '新闻' in task_name:
        owner = 'work-claw'
        task_name = f"work-claw {task_name}"
    elif '提醒' in task_name or '心跳' in task_name:
        # 根据名称中的 agent 确定归属
        if 'dev-claw' in task_name:
            owner = 'dev-claw'
            task_name = f"dev-claw {task_name}"
        elif 'daughter' in task_name:
            owner = 'daughter'
            task_name = f"daughter {task_name}"
        elif 'work-claw' in task_name:
            owner = 'work-claw'
            task_name = f"work-claw {task_name}"
        else:
            owner = 'main'
            task_name = f"main {task_name}"
    else:
        owner = 'main'
```

### **任务类型与归属规则**

| 任务类型 | 归属 Agent | 示例 |
|---------|-----------|------|
| **日报类** | work-claw | AI 新闻日报、中东战况日报、需求挖掘日报 |
| **天气预报** | work-claw | 晨间天气预报、晚间天气预报 |
| **心跳任务** | 对应 Agent | dev-claw 心跳、work-claw 心跳、daughter 心跳 |
| **提醒类** | main/对应 Agent | 工作日志提醒-main、成长日志提醒-daughter |
| **系统任务** | main | 任务清单生成、记忆索引更新、周报生成 |

## 📤 **任务移交规范**

### **移交时更新任务名称**

**文件**: `backend/main.py` - `create_handover()`

**逻辑**：
```python
# 更新任务名称中的 agent 前缀
old_agent = handover_data["from_agent_id"]
new_agent = handover_data["to_agent_id"]

import re
old_prefix_pattern = rf'^{re.escape(old_agent)}[\s-]+'

if re.match(old_prefix_pattern, task.task_name, re.IGNORECASE):
    # 替换前缀为新 agent
    task.task_name = re.sub(old_prefix_pattern, f'{new_agent} ', task.task_name, flags=re.IGNORECASE)
    task.agent_id = new_agent
```

### **移交示例**

**场景**：work-claw 将任务移交给 daughter

**移交前**：
```
任务名：work-claw AI 新闻日报 - 每日早上 7 点
执行者：work-claw
```

**移交后**：
```
任务名：daughter AI 新闻日报 - 每日早上 7 点
执行者：daughter
```

## 📊 **数据一致性保证**

### **cron_sync 同步逻辑**

**文件**: `backend/cron_sync.py`

**每次同步时**：
1. 读取 Cron 任务列表
2. 自动添加/更新任务名称中的 agent 前缀
3. 更新数据库中的 `task_name` 和 `agent_id`
4. 确保任务归属与名称一致

**代码**：
```python
# 总是更新任务名称（确保使用带 agent 前缀的名称）
existing.task_name = cron_task['task_name']

# 更新执行者为归属 Agent
existing.agent_id = owner
```
