## 📋 任务分类

### 三类任务

| 类型 | 说明 | 显示规则 | 示例 |
|------|------|----------|------|
| **普通任务** | 手动创建的任务 | 显示今日创建的 | "dev-claw 心跳" |
| **Cron 任务** | 定时任务 | 显示今日应执行的 | "每日进化日志提醒" |
| **一次性任务** | 项目/自动化任务 | 显示今日创建或正在执行的 | "项目开发 - XXX 功能" |

## 🔍 任务类型识别

### 识别规则

```python
def classify_task_type(task: TaskRecord) -> str:
    """
    分类任务类型
    
    规则:
    1. Cron 任务：task_id 包含 "cron-"
    2. 一次性任务：包含特定关键词（项目、自动化、临时、特别、紧急）
    3. 普通任务：默认
    """
    task_id = task.task_id or ""
    task_name = task.task_name.lower()
    
    # Cron 任务
    if task_id.startswith('cron-'):
        return 'Cron 任务'
    
    # 一次性任务
    one_time_keywords = ['项目', '自动化', '临时', '特别', '紧急']
    if any(kw in task_name for kw in one_time_keywords):
        return '一次性任务'
    
    # 普通任务
    return '普通任务'
```

### 示例

| task_id | task_name | 任务类型 |
|---------|-----------|---------|
| `cron-xxx-xxx` | 每日进化日志提醒 | Cron 任务 |
| `cron-xxx-xxx` | 晨间天气预报 - 每日 6:30 | Cron 任务 |
| `task-001` | dev-claw 心跳 | 普通任务 |
| `task-002` | 项目开发 - 龙虾办公室 | 一次性任务 |
| `task-003` | 自动化脚本测试 | 一次性任务 |
| `task-004` | 临时任务 | 一次性任务 |

## 📅 今日任务过滤规则

### 过滤逻辑

```
显示今日任务
  ↓
过滤周期性任务
  ├─ 每周 X 的任务 → 仅今日是周 X 时显示
  └─ 每月 X 日的任务 → 仅今日是 X 日时显示
  ↓
返回过滤后的任务列表
```

### 规则 1: 每周 X 的任务

**规则**: ❌ 不显示每周 X 的任务（除非今日是周 X）

| 任务名称 | 今日是周一 | 今日是周三 | 今日是周五 |
|---------|----------|----------|----------|
| 每周一例会 | ✅ 显示 | ❌ 隐藏 | ❌ 隐藏 |
| 每周三技术分享 | ❌ 隐藏 | ✅ 显示 | ❌ 隐藏 |
| 每周五复盘 | ❌ 隐藏 | ❌ 隐藏 | ✅ 显示 |

**代码实现**:
```python
# 每周 X 的任务
if '每周一' in task_name and now.weekday() != 0:
    should_show = False
elif '每周二' in task_name and now.weekday() != 1:
    should_show = False
# ... 以此类推
```

---

### 规则 2: 每月 X 日的任务

**规则**: ❌ 不显示每月 X 日的任务（除非今日是 X 日）

| 任务名称 | 今日是 1 日 | 今日是 15 日 | 今日是 20 日 |
|---------|----------|----------|----------|
| 每月 1 日月报 | ✅ 显示 | ❌ 隐藏 | ❌ 隐藏 |
| 每月 15 日总结 | ❌ 隐藏 | ✅ 显示 | ❌ 隐藏 |
| 每月 20 日复盘 | ❌ 隐藏 | ❌ 隐藏 | ✅ 显示 |

**代码实现**:
```python
# 每月 X 日的任务
if '每月 1 日' in task_name and now.day != 1:
    should_show = False
elif '每月 15 日' in task_name and now.day != 15:
    should_show = False

# 匹配"每月 X 日"格式
import re
monthly_match = re.search(r'每月 (\d{1,2}) 日', task_name)
if monthly_match:
    target_day = int(monthly_match.group(1))
    if now.day != target_day:
        should_show = False
```

---

### 规则 3: 每日 X 点的任务

**规则**: ✅ 今日全部显示（不管时间是否已过）

| 任务名称 | 当前时间 | 显示状态 |
|---------|---------|---------|
| 每日 6:30 晨间预报 | 08:00 | ✅ 显示 |
| 每日 7:00 AI 新闻 | 09:00 | ✅ 显示 |
| 每日 22:00 晚间总结 | 23:00 | ✅ 显示 |

**说明**: 每日任务不限制时间范围，全部显示。

## 📊 过滤流程图

```
开始
  ↓
查询今日创建的任务
  ↓
遍历每个任务
  ↓
是每周 X 的任务？
  ├─ 是 → 今日是周 X 吗？
  │        ├─ 是 → 显示 ✅
  │        └─ 否 → 隐藏 ❌
  └─ 否 ↓
是每月 X 日的任务？
  ├─ 是 → 今日是 X 日吗？
  │        ├─ 是 → 显示 ✅
  │        └─ 否 → 隐藏 ❌
  └─ 否 ↓
显示任务 ✅
  ↓
返回过滤后的列表
```

## 🎯 最佳实践

### 1. 任务命名规范

**推荐**:
- ✅ `每周一例会` - 清晰的周期性
- ✅ `每日进化日志提醒` - 清晰的频率
- ✅ `项目开发 - XXX 功能` - 清晰的一次性

**不推荐**:
- ❌ `例会` - 无周期性信息
- ❌ `日志` - 无频率信息
- ❌ `临时` - 太模糊

### 2. 过滤逻辑

- ✅ 先查询今日任务
- ✅ 再过滤周期性任务
- ✅ 返回过滤后的列表

### 3. 任务类型识别

- ✅ 优先识别 Cron 任务（task_id 前缀）
- ✅ 再识别一次性任务（关键词）
- ✅ 默认普通任务
